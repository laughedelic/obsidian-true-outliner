* first, cosmetic: bullets have to be more prominent. we want the real outliner experience, remember? don't be subtle about it.
* next, it looks like the code blocks don't get indented at all.
* headings don't get indented. which they should, according to their level relative to the parent.

* the hierarchical structure in outline mode has to be very clear and non-ambiguous. that's the whole point. clear, consistent indentation, visible prominent node markers.
